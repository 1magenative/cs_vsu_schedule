# 🎓 CSF VSU Schedule Bot

![Python](https://img.shields.io/badge/python-3.11+-blue.svg)
![Aiogram](https://img.shields.io/badge/aiogram-3.x-orange.svg)
![FastAPI](https://img.shields.io/badge/FastAPI-modern-green.svg)
![SQLite](https://img.shields.io/badge/SQLite-light-lightgrey.svg)

Современный и быстрый Telegram-бот для просмотра расписания факультета компьютерных наук ВГУ. Включает в себя интерактивное **Mini App** приложение с обратным отсчетом до начала/конца пар.

## ✨ Основные возможности

### 🤖 Бот
*   **Умный поиск:** Находите расписание любого преподавателя по всему факультету.
*   **Гибкие режимы:** Три режима отображения (`обычный`, `po***` и `пивко`), меняющие скучные «Нет пары» на что-то более душевное.
*   **Учет четности недель:** Автоматическое определение Числителя/Знаменателя.
*   **Удобная навигация:** Кнопки для расписания на Сегодня, Завтра и всю Неделю.

### 📱 Mini App
*   **Интерактивный интерфейс:** Плавное веб-приложение прямо внутри Telegram.
*   **Виджет обратного отсчета:** Показывает в реальном времени, сколько осталось до конца текущей пары или начала следующей.
*   **Синхронизация времени:** Время в приложении всегда совпадает с временем сервера, независимо от настроек вашего телефона.
*   **Умная фильтрация:** Автоматически скрывает пустые окна и завершенные учебные дни.

### ⚙️ Админ-панель
*   **Управление временем:** Настройка часового пояса сервера кнопками `+1ч` / `-1ч` прямо в боте.
*   **Статистика:** Мониторинг общего количества пользователей и активных за текущие сутки.
*   **Рассылки:** Возможность отправить важное объявление всем пользователям одной кнопкой.

## 🚀 Технологический стек

*   **Backend:** Python 3.11+ (asyncio)
*   **Telegram API:** [aiogram 3.x](https://github.com/aiogram/aiogram)
*   **Web API:** [FastAPI](https://fastapi.tiangolo.com/) + Uvicorn
*   **Frontend:** Vanilla JS, CSS3 (Glassmorphism), HTML5
*   **Database:** SQLite (aiosqlite)
*   **Scheduler:** APScheduler (для автоматической смены типа недели)

## 🛠 Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/yourusername/cs-vsu-schedule-app.git
    cd cs-vsu-schedule-app
    ```

2.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Настройте переменные окружения:**
    Создайте файл `.env` в корневой папке:
    ```env
    BOT_TOKEN=ваш_токен_бота
    ADMIN_ID=ваш_телеграм_id
    WEBAPP_URL=https://your-domain.com
    REPORTS_CHAT_ID=id_группы_для_жалоб
    ```

4.  **Запустите бота:**
    ```bash
    python main.py
    ```

## 📋 Структура проекта

*   `main.py` — Ядро бота и серверная часть.
*   `database.py` — Асинхронная работа с БД SQLite.
*   `parser.py` — Логика извлечения данных из расписания.
*   `normalize.py` — Скрипт для обработки Excel-файла с расписанием.
*   `webapp/` — Файлы Telegram Mini App (HTML, CSS, JS).
*   `bot_database.db` — Локальная база данных (создается автоматически).
